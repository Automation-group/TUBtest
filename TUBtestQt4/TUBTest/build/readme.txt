Для сборки требутеся:
 - Qt4 последней версии
 - Cmake версии 2.8 или выше
 - QWT 5.2.3
 - MinGW 4.8.4(windows), G++ 4.8.4(linux)

=========================================================================
Сборка проекта под Linux
>> ./build.sh
* файл build.sh нужно сделать исполняемым

=========================================================================
Сборка проекта под Windows
>> build.bat

 - для сборки проекта под windows необходимо скопировать исходники qwt-5.2.3 в корень проекта
 - папку qextserialport нужно удалить, а qextserialport_for_windows
переименовать в qextserialport
 - в файлах MainWindow.cpp и SearchDevices.cpp закоментировать строки // for linux и раскомментировать строки // for windows
 - для работы собранной программы необходимо скопировать требующиеся библиотеки .dll в папку /build
